# Iris Dataset - Data Cleaning & Preprocessing

This project demonstrates data cleaning and preprocessing on the Iris dataset.

---

## Dataset

- Source: [Iris Dataset](https://archive.ics.uci.edu/ml/datasets/iris)
- 150 rows, 6 columns (Id, SepalLengthCm, SepalWidthCm, PetalLengthCm, PetalWidthCm, Species)
